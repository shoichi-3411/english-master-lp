# ランディングページ修正内容

## 実装日時
2026年1月11日

## 修正内容

### 1. モバイルファースト設計への変更

#### ルート CSS 変数の調整
- **基本フォントサイズ**: 18px → 16px
- **大見出し (h1)**: 48px → 32px
- **見出し (h2)**: 36px → 28px
- **サブ見出し (h3)**: 28px → 22px
- **スペーシング**: 全体的に削減

#### デスクトップ対応
- `@media (min-width: 1024px)` で元のサイズに戻す

### 2. セクションタイトル (section-title) の改善

```css
.section-title {
  margin-bottom: var(--spacing-md);  /* 削減 */
  padding-bottom: var(--spacing-md);  /* 削減 */
  line-height: 1.3;  /* 改善 */
  word-break: break-word;  /* 追加 */
  max-width: 95%;  /* 追加 */
}
```

**効果**: 見出しの改行位置が改善され、読みやすくなりました。

### 3. 統計数字 (summary-stats) の均等配置

```css
.summary-stats {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: var(--spacing-md);
  margin-top: var(--spacing-md);  /* 削減 */
  width: 100%;
}

.stat-card {
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
}

.stat-value {
  font-size: var(--font-size-2xl);
  line-height: 1.2;
}

.stat-label {
  font-weight: 600;
}
```

**効果**: 「250点」「97%」などの統計数字が均等に配置されます。

### 4. エグゼクティブプランの「推奨」バッジ修正

```css
.plan-badge {
  position: absolute;
  top: 10px;  /* -12px から変更 */
  right: 10px;  /* 20px から変更 */
  background-color: var(--color-gold);
  color: var(--color-navy);
  padding: 6px 12px;
  border-radius: 20px;
  font-weight: 700;
  font-size: 12px;
  white-space: nowrap;
  z-index: 20;  /* 追加 */
}
```

**効果**: バッジがカード内に配置され、重なりが解消されました。

### 5. 価格表示の改行位置最適化

```css
.plan-original,
.plan-final {
  display: flex;
  justify-content: space-between;
  align-items: center;
  margin-bottom: var(--spacing-md);
  flex-wrap: wrap;  /* 追加 */
}

.plan-original .price,
.plan-final .price {
  white-space: nowrap;  /* 追加 */
  margin-left: var(--spacing-sm);  /* 追加 */
}
```

**効果**: 「特別価格」「円」などの単語が適切に改行されます。
