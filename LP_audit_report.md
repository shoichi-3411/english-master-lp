## 2. 重大な指摘（致命的）上位5

### 指摘1: 1stビューのCTA配置が不明確（PR-01違反）
**問題点**: ヒーロー内にCTAボタンが見当たらない。「詳細を見る」はスクロール促進だが、購買行動への直接導線がない。

**修正案**:
- ヒーロー内に「今すぐ申し込む」ボタンを追加（スタンダードプランへの直接リンク）
- 「詳細を見る」は補助CTA（スクロール促進）として階層化

**重要度**: 🔴 致命的

---

### 指摘2: 画像プレースホルダーの多用（PR-07違反）
**問題点**: 「【受講生のビフォーアフター画像挿入位置】」が複数箇所に存在。架空画像が未実装のため、信頼性が低下。

**修正案**:
- 架空の受講生ビフォーアフター画像を生成・配置
- 講師の顔写真プレースホルダーを実装

**重要度**: 🔴 致命的

---

### 指摘3: フォーム要件が不明確（PR-10違反）
**問題点**: CTA「申し込む」をクリックした後のフォーム設計が不明。必須項目、入力補助、エラーハンドリングが定義されていない。

**修正案**:
```
必須項目:
- 名前（テキスト）
- メールアドレス（email）
- 電話番号（tel）
- 選択プラン（ラジオボタン：スタンダード/エグゼクティブ）

入力補助:
- メールアドレス例を表示
- プラン選択時に価格を即座に表示

エラーハンドリング:
- 必須項目未入力時：「〇〇は必須です」とテキスト表示
- メール形式エラー：「正しいメール形式で入力してください」
```

**重要度**: 🔴 致命的

---

### 指摘4: アクセシビリティ要件が未実装（PR-16違反）
**問題点**: 
- コントラスト比の検証なし（PR-12）
- タップ領域サイズが不明（PR-11）
- キーボード操作時のフォーカス可視性が不明（SC 2.4.7）

**修正案**:
- 本文テキストのコントラスト比を4.5:1以上に確保
- ボタンのタップ領域を最小48×48px（推奨）に設定
- フォーカス時に明確な視覚フィードバック（アウトライン/背景色）を追加

**重要度**: 🔴 致命的

---

### 指摘5: 計測イベント設計が未実装（PR-15違反）
**問題点**: Google Analytics等の計測イベントが定義されていない。ボトルネック分析ができない。

**修正案**:
```
最低限のイベント:
1. hero_cta_click（ヒーロー内CTA）
2. main_cta_click（各セクション内CTA）
3. form_start（フォーム開始）
4. form_submit_success（フォーム送信成功）
5. scroll_to_section（重要セクション到達）
```

**重要度**: 🔴 致命的

## 3. 重要な改善点 上位10

### 改善1: 「3段階マスターシステム」の説明が不十分
**問題点**: Stage 1/2/3の説明が短く、各ステージで何を学ぶかが不明確。

**修正案**:
```
Stage 1: 基礎固め（0-30日）
- ビジネス英語コアフレーズ500習得
- リスニング力の基礎構築
- 実践的な発音トレーニング
到達目標: TOEIC平均80点アップ、基本的なビジネス会話が可能

Stage 2: 実践力養成（31-60日）
- 会議でのプレゼンテーション練習
- メール・レポート作成スキル
- 交渉・商談シミュレーション
到達目標: TOEIC平均90点アップ、実務会話が可能

Stage 3: 応用・定着（61-90日）
- 複雑なビジネス状況への対応
- 業界特化表現の習得
- 総合実践トレーニング
到達目標: TOEIC平均80点アップ、即戦力レベルに到達
```

**重要度**: 🟠 重要

### 改善5: 固定CTA（スティッキーボタン）の実装（PR-09）

### 改善6: 「1週間で4,000時間分の学習効果」の根拠が不明確
**問題点**: 数字が提示されているが、計算根拠が不明。曖昧表現。

**修正案**:
```
「1週間で4,000時間分の学習効果」の根拠を明記:
- 従来型スクール: 週2時間 × 年50週 = 年100時間
- 本プログラム: 週3.5時間 × 年50週 = 年175時間
- 学習効率: 本プログラムは従来型の2.5倍
- 3ヶ月での効果: 従来型の3年分に相当
```

**重要度**: 🟠 重要

```
返金保証の条件:

```

**重要度**: 🟠 重要

**重要度**: 🟠 重要

## 6. 根拠不足の指摘

| 項目 | 根拠が必要な理由 | 追加すべき出典タイプ |
|------|-----------------|-------------------|
| 「3ヶ月で実務レベルに到達」 | 学習科学的根拠が不明 | 言語習得研究、SLA（Second Language Acquisition）論文 |
| 「1日30分で効果が出る」 | 学習時間と効果の相関が不明 | 認知心理学、スパーシング効果の研究 |
| 「TOEIC平均250点アップ」 | 統計的有意性が不明 | 標本数、標準偏差、信頼区間の明記 |
| 「継続率97%」 | 業界平均との比較が不明 | 英会話スクール業界の平均継続率データ |
| 「従来型スクール比2.5倍の効率」 | 計算根拠が不明 | 学習効率の定義、計測方法の明記 |
